# Link Cleaner

Part of the Fridge Combo project. A high-strength URL shredder for those who want absolute minimalism and privacy.

## 1 Tool / 1 Action

Shred the clutter. **Link Cleaner** aggressively removes everything after the `?` in any URL. No exceptions, no mercy.

## Philosophy
* **1 Tool, 1 Action.**
* **High-Strength:** It doesn't "filter"; it "shreds".
* **No Login / No Ads / No Tracking**

## How to Use
1. **Paste:** Paste any long, messy URL.
2. **Instant Shred:** The tool immediately cuts off all parameters.
3. **Copy:** Tap [ COPY SHREDDED LINK ] to get the shortest possible URL.

## ⚠️ CAUTION
This is a "high-strength" tool. Removing all parameters may break links that rely on them (like YouTube video IDs or search queries). For a safer, smart cleaning experience, use **#4 Pure Link**.
